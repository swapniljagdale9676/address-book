# Address-Book_Project

A command-line based Address Book project developed in C that helps users securely store, search, edit, and manage contact details efficiently. This project demonstrates core C programming concepts such as structures, file handling, modular programming, and build automation using Makefile.

## 📌 Key Features

* Add new contacts
* Search contacts by name or phone number
* Edit existing contact details
* Delete contacts
* Display all saved contacts
* Persistent data storage using file handling
* Command-line based execution
* Robust input validation
* Modular code structure using header files
* Built using Makefile

## 🛠 Technologies & Concepts

* C Programming

## ⚙️ Build Instructions

Ensure `make` is installed, then run:

make

This will compile the project and generate the executable file.

To clean build files:

make clean

## ▶️ Usage

Run the executable:

./AddressBook

(Replace `AddressBook` with your executable name if different.)
